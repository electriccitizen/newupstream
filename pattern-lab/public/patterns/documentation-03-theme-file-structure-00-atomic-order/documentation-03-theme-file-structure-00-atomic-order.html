Atomic Design Principles

Citizen Patterns is an uses an atomic design approach to theming. Atomic design is a system for developing a component-based design from the smallest element to the largest.

The Atomic Order:

  • Atoms
  • Molecules
  • Organisms
  • Templates
  • Pages

This theme defines global elements like color, fonts, macros and mixins, and builds them into text, links, icons, buttons, etc. These elements are built into blocks and menus. More advanced elements are widgets and the site header and footer. Everything is combined at end into layouts, pages and nodes.

Citizen Patterns Atomic File Structure

The Citizen Atomic Order:

  • Global
    1. Colors
    2. Fonts
    3. Grid
    4. Mixins
    5. Macros
    6. Layouts
    7. Global Scss files
  • Atoms
    1. Links
    2. Text
    3. Lists
    4. Images
    5. Forms
    6. Buttons
    7. Tables
  • Molecules
    1. Blocks
    2. Card
    3. Menus
    4. Pagers
  • Organisms
    1. Card Grid
    2. Widgets (paragraphs)
    3. Site (header & Footer)
  • Pages
    1. Layouts
    2. Nodes
    3. Search
    4. Taxonomy
    5. Pages.scss (custom, non-node, front pages)
  • Site Wide
    1. Global High Level
    2. Drupal Base Elements (media, views, etc)
    3. Styleguide-specfic.scss
