Font Awesome Integration

Font Awesome is an icon library that is called via CDN in the global library in citizen_patterns.libraries.yml. The version that should be called for each site depends on the icons defined in the design. The default version for base icons is FontAwesome 5. Some designs call for a switch to 4.7

Font Awesome icons can used in different ways:

  1. The prefer method is bring in through an icon tag: <i class="fa fa-angle-right">. NOTE: the preceding 'fa' in that tag can also be fas, far, or fal depending on what version of Font Awesome you are using (see docs).
  2. You can also add icons through css pseudo elements (:before, :after) and the FA mixins the theme provides.
    Example:
    .selector:before{@include faSolid($fa-times-circle, 20px);}
